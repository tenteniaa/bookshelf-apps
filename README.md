# Bookshelf Apps
 
 Aplikasi ini merupakan media yang dapat digunakan untuk mencatat buku-buku yang telah dibaca maupun yang akan dibaca. 
 Aplikasi ini akan menyimpan histori bacaan ke dalam cache server.
